### 11.2.2    关键字类型的要求

*默认情况下，标准库使用关键字类型的 < 运算符来比较两个关键字*

- #### 有序容器的关键字

  - 想要自定义关键字需要定义一个**严格弱序列：**

    - 两个关键字不能同时 “小于等于对方”
    - 如果 k1 <= k2，且 k2 <= k3，那么 k1 必须 <= k3
    - 如果存在两个关键字，任何一个都不 “小于等于” 另一个，那么称这两个关键字为等价
